WEB

The World Wide Web also called WWW or Web or W3 delivers what you see right now .... The World Wide Web (or simply the "Web") is a system of interlinked, hypertext documents that runs over the Internet. With a Web browser, a user views Web pages that may contain text, images, and other multimedia and navigates between them using hyperlinks (Wikipedia, retrieved 23:25, 28 March 2007 (MEST)).
